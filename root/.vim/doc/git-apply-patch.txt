See: ~/.vim/doc/git-apply-patch.md

*git-format-patch*
生成patch ~
>
    git format-patch <commit_hash>
    git format-patch -1 <commit_hash>
<

*git-apply-patch*
应用patch ~
>
    git apply /path/to/my_changes.patch
<

*git-am*
应用patch和原本的提交信息（可以加`--signoff`）~
>
    git am /path/to/patch-file.patch
<
